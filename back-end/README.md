# Back-end da Api

Back end responsável por gerência os end-points da api. Foi refeito utilizando o GraphQL.

## Requisitos

Requisitos minimo para rodar o back-end da aplicação:

- NodeJs v16 ou superior

## Tecnologias usadas

- NodeJs
- GraphQL
- ApolloServer
- Jsonwbtoken
- Bcryptjs
- Prisma

## Funcionalidades

- Craia conta de usuário
- Realizer login
- Gerar token de autenticação
- Cadastrar favoritos
- Busca todos favoritos
- Apagar favoritos

## Rodando localmente

Entre na pasta do servidor

```bash
  cd back-end
```

Inicie o servidor

```bash
  npm run dev
```
